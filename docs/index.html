The Reproduction Method of Deep Q-network
with DQN3.0-level Performance

Shigeru Matsumura
Ver 1.0: 1 June 2018

It is very easy way to reproduce a Deep Q-network (DQN) (Mnih et al., 2015) with the same performance as DQN3.0 (DeepMind, 2017), just create it in the same creation method as DQN3.0. But as far as I know, there is nowhere to explanation of easiest way as above, and also its implementation.  So I studied the reproduction method of DQN in DQN3.0, and succeeded reproduce some version of DQNs with DQN3.0-level performance (Figure 1).  This article will explained the reproduction method of DQN in detail and how to implement a DQN with DQN3.0-level performance using Python with major deep learning frameworks such as PyTorch, MXNet, Tensorflow and CNTK (the last two uses Keras).  After that, proves that reproducing DQN3.0 with other languages and deep learning frameworks is the same as we can reproduce $1+1=2$ with any programming language and deep learning framework.
